# BugMinion

***BugMinion*** is a simple bug themed tic tac toe game.

You have 5 seconds to play.

Move with the _arrow_ or _wasd_ keys and hit _enter_ to lay your bug.

You need to have `numpy` and `pygame` installed:
```
pip3 install numpy
pip3 install pygame
```
